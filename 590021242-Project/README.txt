**1. Introduction:**
The Math Toolkit Web Application is a Python-based project developed using the Flask framework. It provides a user-friendly web interface to perform various mathematical operations such as factorial calculation, Fibonacci series generation, prime number checking, palindrome detection, and simple interest computation.

This project demonstrates the application of core programming concepts in a real-world scenario using a web-based approach instead of a traditional command-line interface.

---

**2. Objectives:**

* To develop a web-based mathematical toolkit using Python
* To implement mathematical operations using functions and logic
* To create an interactive and user-friendly interface
* To understand the integration of frontend and backend technologies

---

**3. Technologies Used:**

* Python (Flask Framework)
* HTML
* CSS

---

**4. Features of the Application:**

* Factorial calculation
* Fibonacci series generation
* Prime number checking
* Palindrome number detection
* Simple interest calculation
* Interactive and responsive web interface

---

**5. Working of the Project:**
The application uses Flask as the backend server to process user inputs received from the web interface. The user selects an operation and provides the required input values. The request is sent to the Flask server, where the corresponding function is executed. The result is then returned and displayed on the web page dynamically.

**9. How to Run the Project:**

1. Install Flask using: pip install flask
2. Open terminal in project folder
3. Run the application using: python app.py
4. Open browser and go to: http://127.0.0.1:5000
